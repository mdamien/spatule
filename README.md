#Spatule

Small django project for an user-maintened website

##Installation & test 
Install the depencies  
Complete ROOT_DIR in settings.py  
Run the server 
> ./manage.py runserver
Go to localhost:8000

There is already a bdd with users & data  
Modify the data by going to /admin:  
Super-admin account: tic/t  
User account: webmaster/test  

##Dependencies

sudo apt-get install python-django python-sqlite 

pip install django-tinymce
